# 地图跳转缩放优化

## 🔧 问题描述
搜索用户后，地图会跳转到用户位置，但缩放级别被强制调整到15级，导致视野太近，用户需要手动缩小地图才能看清周围环境。

## ✨ 解决方案
修改 `jumpToUserLocation` 函数，**只改变地图中心点，保持当前缩放级别不变**。

## 📝 代码修改

### 修改前 (问题代码)
```javascript
map.setCenter([lng, lat]);
map.setZoom(15); // 强制设置缩放级别为15
```

### 修改后 (优化代码)
```javascript
map.setCenter([lng, lat]);
// 保持当前缩放级别不变，只改变中心点
```

## 🎯 优化效果

| 功能 | 修改前 | 修改后 |
|------|--------|--------|
| 地图中心 | ✅ 跳转到用户位置 | ✅ 跳转到用户位置 |
| 缩放级别 | ❌ 强制15级（太近） | ✅ 保持当前级别 |
| 用户体验 | ❌ 需手动缩小地图 | ✅ 直接看到合适视野 |

## 🧪 测试方法
1. 访问：`http://localhost:5001/main?customerId=1`
2. 调整地图到合适的缩放级别
3. 搜索并选择员工
4. 观察：地图中心跳转到员工位置，但缩放级别保持不变
