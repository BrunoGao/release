# LJWX-BigScreen 用户使用文档

## 文档概述

本文档面向LJWX-BigScreen智能健康数据分析平台的最终用户，提供详细的系统使用指南、功能介绍和操作说明。

## 1. 系统简介

### 1.1 平台概述

LJWX-BigScreen是一个专业的智能健康数据分析平台，提供实时健康监测、数据分析和可视化展示功能。平台支持多种可穿戴设备的数据接入，通过先进的算法对用户健康状况进行评估和预警。

### 1.2 主要功能

- **实时健康监测**：24小时不间断监测心率、血氧、体温等健康指标
- **智能健康评分**：基于9维健康指标的综合评分算法
- **可视化大屏**：直观的数据展示和3D地图可视化
- **智能告警系统**：多级告警机制和实时消息推送
- **趋势分析**：历史数据分析和健康趋势预测
- **设备管理**：统一的设备注册、绑定和状态管理

### 1.3 支持的设备类型

- 智能手表（心率、步数、睡眠监测）
- 健康监测仪（血压、血氧、体温）
- 环境传感器（温度、湿度、空气质量）
- 专业医疗设备（血糖仪、体重秤等）

#### 2.1.1 访问地址

通过浏览器访问系统地址：`http://your-domain.com/main?customerId=YOUR_CUSTOMER_ID`

## 3. 功能详细说明

### 3.1 实时统计面板

#### 3.1.1 数据概览

实时统计面板显示以下关键指标：

**健康数据**
- **当前值**：当日已采集的健康数据条数
- **趋势指标**：与昨日同期对比的变化百分比
- **状态指示**：🟢正常 🟡注意 🔴异常

**待处理告警**
- **紧急告警**：需要立即处理的严重告警
- **一般告警**：需要关注的普通告警
- **告警趋势**：告警数量变化趋势

**活跃设备**
- **在线设备**：当前正常工作的设备数量
- **离线设备**：超过5分钟未上报数据的设备
- **设备健康度**：设备整体运行状态评估

**未读消息**
- **系统消息**：系统自动生成的通知消息
- **告警消息**：来自告警系统的重要提醒
- **消息优先级**：🔴紧急 🟡重要 🔵一般

#### 3.1.2 日期选择功能

在实时统计面板的右上角，您可以：

1. **选择查看日期**：点击日期选择器选择特定日期的数据
2. **对比分析**：系统自动与前一天的数据进行对比
3. **快速切换**：使用快捷键或预设选项快速切换时间范围

#### 3.1.3 系统状态指示器

页面底部的系统状态指示器显示：
- **🟢 系统正常**：所有服务运行良好
- **🟡 部分异常**：某些功能可能受影响
- **🔴 系统故障**：需要技术支持介入

### 3.2 人员管理面板

#### 3.2.1 人员统计信息

**总体统计**
- **总人数**：当前组织内的总人数
- **已绑定设备**：已成功绑定可穿戴设备的人员数量
- **绑定率**：已绑定设备人员占总人数的百分比

**活跃度统计**
- **活跃部门**：有人员在线活动的部门数量
- **在线人员**：当前在线（设备活跃）的人员数量
- **绑定设备**：总的已绑定设备数量
- **告警人员**：当前有健康告警的人员数量

#### 3.2.2 部门分布图表

左侧的部门分布饼图显示：
- 各部门人员数量占比
- 不同部门的颜色标识
- 鼠标悬停显示具体数值
- 点击图例可隐藏/显示特定部门

#### 3.2.3 在线状态图表

右侧的在线状态图表显示：
- **在线**：设备正常上报数据
- **离线**：超过5分钟未收到数据
- **异常**：设备或健康数据异常

#### 3.2.4 人员详情查看

点击"详情 →"按钮可以：
1. 查看详细的人员列表
2. 搜索特定人员
3. 查看个人健康数据
4. 管理设备绑定状态

中间的3D地图区域提供：
- **实时位置显示**：显示人员当前位置

- **右键旋转**：旋转3D视角

**快捷键操作**
```

方向键    - 微调地图位置
+/-      - 缩放地图
R        - 随机切换视角
```

1. **打开筛选面板**

**图表类型**

- **区域图**：显示指标数值的波动范围

**支持的健康指标**

### 3.6 健康评分面板

#### 3.6.1 综合健康评分

**评分展示**
- **评分环形图**：直观显示当前健康评分（0-100分）
- **评分等级**：优秀(90-100)、良好(80-89)、一般(70-79)、注意(60-69)、风险(<60)
- **评分趋势**：与历史评分的对比

**评分构成**
健康评分基于以下9个维度：
1. **心率**（权重：1.5）- 正常范围：60-100 bpm
2. **血氧**（权重：1.2）- 正常范围：95-100%
3. **体温**（权重：1.0）- 正常范围：36.3-37.2°C
4. **收缩压**（权重：1.3）- 正常范围：90-140 mmHg
5. **舒张压**（权重：1.3）- 正常范围：60-90 mmHg
6. **压力指数**（权重：1.1）- 正常范围：0-70
7. **步数**（权重：0.8）- 正常范围：6000-10000步
8. **运动距离**（权重：0.7）- 正常范围：3-8公里
9. **睡眠时长**（权重：2.0）- 正常范围：7-9小时

#### 3.6.2 健康雷达图

**雷达图说明**
- **蓝色区域**：当前健康状态
- **透明区域**：理想健康状态
- **各个轴线**：代表不同健康指标
- **数值范围**：0-100分制

**如何解读雷达图**
- 面积越大：整体健康状况越好
- 形状规则：各指标发展均衡
- 凹陷部位：需要重点关注的健康指标
- 突出部位：表现优异的健康指标

#### 3.6.3 健康建议

系统根据您的健康数据自动生成个性化建议：

**运动建议**
- 步数不足时：建议增加日常散步时间
- 运动过量时：建议适当休息和恢复

**作息建议**
- 睡眠不足时：建议调整作息时间
- 压力过大时：建议进行放松训练

**饮食建议**
- 血压异常时：建议控制盐分摄入
- 体重超标时：建议调整饮食结构

### 3.7 告警信息面板

#### 3.7.1 告警级别分类

**严重告警（红色）**
- 心率异常（>120 bpm或<50 bpm）
- 血氧过低（<90%）
- 体温异常（>38°C或<35°C）
- 血压严重异常

**重要告警（橙色）**
- 轻微生理指标异常
- 设备电量不足
- 长时间无运动

**一般告警（黄色）**
- 数据异常提醒
- 设备离线通知
- 定期检查提醒

#### 3.7.2 告警处理流程

**自动处理**
1. 系统检测到异常数据
2. 自动评估告警级别
3. 发送通知给相关人员
4. 记录告警历史

**手动处理**
1. 查看告警详情
2. 确认告警原因
3. 采取相应措施
4. 标记告警状态

#### 3.7.3 告警统计图表

**告警类型分布**
- 显示不同类型告警的数量分布
- 饼图形式展示占比情况

**告警级别统计**
- 统计各个级别的告警数量
- 柱状图显示严重程度分布

**告警趋势分析**
- 显示告警数量的时间趋势
- 帮助识别问题发生规律

### 3.8 消息信息面板

#### 3.8.1 消息类型

**健康提醒**

### 4.4 告警配置

#### 4.4.1 个人告警阈值设置

用户可以根据个人情况调整告警阈值：

**心率告警**
```
严重高心率：> 120 bpm
一般高心率：> 100 bpm
一般低心率：< 60 bpm
严重低心率：< 50 bpm
```

**血氧告警**
```
严重缺氧：< 90%
轻度缺氧：< 95%
```

**体温告警**
```
高热：> 38.5°C
发热：> 37.5°C
低体温：< 35°C
```

### 5.3 数据理解问题

#### Q7：健康评分是如何计算的？

**A7：评分计算说明：**

**计算公式：**
```
健康评分 = Σ(单项指标得分 × 权重) / Σ权重

其中单项指标得分计算：
- 正常范围内：85-100分
- 轻微异常：60-84分  
- 明显异常：30-59分
- 严重异常：0-29分
```

**权重分配：**
- 睡眠时长：权重2.0（最重要）
- 心率：权重1.5
- 血压：权重1.3
- 血氧：权重1.2
- 压力指数：权重1.1
- 体温：权重1.0
- 步数：权重0.8
- 运动距离：权重0.7

#### Q8：什么是正常的健康指标范围？

**A8：各项健康指标正常范围：**

```
指标名称        正常范围           单位
──────────────────────────────
心率           60-100            bpm
血氧饱和度     95-100            %
体温           36.3-37.2         °C
收缩压         90-140            mmHg
舒张压         60-90             mmHg
步数           6000-10000        步/天
睡眠时长       7-9               小时/天
压力指数       0-70              -
运动距离       3-8               公里/天
```

**特别说明：**
- 以上范围为一般成年人标准
- 具体标准可能因年龄、性别、体质而异
- 如有特殊情况请咨询医生

### 8.3 常见疾病指标异常模式

#### 8.3.1 心血管疾病

**高血压**
- 收缩压 ≥140 mmHg 或 舒张压 ≥90 mmHg
- 常伴随头痛、眩晕症状
- 需要持续监测和生活方式干预

**心律不齐**
- 心率过快（>100 bpm）或过慢（<60 bpm）
- 心率变异性异常
- 可能伴随心悸、胸闷症状

**冠心病**
- 运动后心率恢复缓慢
- 血压异常波动
- 可能出现胸痛症状

#### 8.3.2 代谢性疾病

**糖尿病**
- 空腹血糖 ≥7.0 mmol/L
- 随机血糖 ≥11.1 mmol/L
- 可能伴随多饮、多尿症状

**肥胖症**
- BMI ≥28 kg/m²（中国标准）
- 体脂率男性>25%，女性>30%
- 常伴随高血压、高血糖

### 8.4 紧急情况处理指南

#### 8.4.1 严重告警处理

**心率严重异常**
```
情况：心率 >140 bpm 或 <40 bpm
处理：
1. 立即停止当前活动
2. 保持冷静，深呼吸
3. 如症状持续>5分钟，立即就医
4. 联系紧急联系人
```

**血氧严重偏低**
```
情况：血氧饱和度 <85%
处理：
1. 确保呼吸道通畅
2. 采取半坐位或坐位
3. 开窗通风，保持空气流通
4. 立即寻求医疗帮助
```

**体温严重异常**
```
情况：体温 >39°C 或 <35°C
高热处理：
1. 物理降温（温水擦拭）
2. 多饮水，注意电解质平衡
3. 避免过度包裹
4. 及时就医

低体温处理：
1. 迅速保暖
2. 避免剧烈运动
3. 饮用温热饮品
4. 寻求医疗帮助
```
