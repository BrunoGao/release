### 技术架构层次

#### 🎨 前端技术栈
- **管理后台**: Vue 3 + TypeScript + Vite (现代化SPA)
- **数据大屏**: Python Flask + Jinja2 + ECharts (服务端渲染)
- **样式系统**: CSS3 + 响应式设计
- **构建工具**: Webpack + Vite

#### ⚙️ 后端技术栈
- **API服务**: Spring Boot 2.x + MyBatis
- **数据处理**: Python Flask + SQLAlchemy
- **数据库**: MySQL 8.0 + Redis 6.x
- **消息队列**: 集成消息中间件

#### 🚀 部署运维栈
- **容器化**: Docker + Docker Compose
- **CI/CD**: 自动化脚本 + Shell
- **监控**: 系统健康检查
- **扩展性**: 微服务架构支持
