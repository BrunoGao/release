## 🎯 核心功能模块

### 🖥️ ljwx-bigscreen - 数据大屏系统
- **功能**: 实时健康数据可视化展示
- **技术**: Python Flask + ECharts + WebSocket
- **特色**: 多租户支持、实时告警、响应式设计

### 🛠️ ljwx-admin - 管理后台系统  
- **功能**: 系统管理、用户权限、设备管理
- **技术**: Vue 3 + TypeScript + Element Plus
- **特色**: RBAC权限体系、动态路由、国际化

### 🔗 ljwx-boot - 后端API服务
- **功能**: RESTful API、业务逻辑、数据持久化
- **技术**: Spring Boot + MyBatis + Redis
- **特色**: 高并发支持、缓存优化、定时任务
