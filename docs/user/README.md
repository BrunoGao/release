## 🚀 快速开始

### 新用户推荐流程

1. **快速入门** - 阅读 [许可证快速入门指南.md](./许可证快速入门指南.md)
2. **详细配置** - 查看 [许可证系统用户操作手册.md](./许可证系统用户操作手册.md)
3. **自动化工具** - 使用 [license-tools.sh](./license-tools.sh) 脚本

### 使用自动化脚本

```bash
# 查看帮助
./docs/user/license-tools.sh help

# 生成许可证
./docs/user/license-tools.sh generate --customer "测试客户" --days 365

# 部署许可证
./docs/user/license-tools.sh deploy

# 验证许可证
./docs/user/license-tools.sh verify
```
