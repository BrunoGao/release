基于项目：[ljwx-boot](https://github.com/brunoGao/ljwx-boot)

```
LjwxBoot Starter
├── ljwx-boot-starter-code-generate -- 代码生成
├── ljwx-boot-starter-common -- 基础模块
├── ljwx-boot-starter-database-mybatis-plus -- MyBatis Plus
├── ljwx-boot-starter-database-mysql -- MySQL 数据库
├── ljwx-boot-starter-database-redis -- Redis 缓存
├── ljwx-boot-starter-database-sharding -- Sharding 分表分库
├── ljwx-boot-starter-job-quartz -- Quartz 调度
├── ljwx-boot-starter-job-oss -- OSS 对象存储
├── ljwx-boot-starter-sa-token -- Sa Token 鉴权
└── pom.xml -- 公共依赖
```
